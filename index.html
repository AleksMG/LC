Математическое обоснование обратимости:

Шифратор использует гарантированно обратимые математические операции:

  • Аффинное преобразование: C = (a×P + b) mod N, обратное: P = a⁻¹×(C - b) mod N
  • Экспоненциальное преобразование: C = K^P mod N, обратное: P = logₖ(C) mod N
  • Матричное преобразование: C = M×P mod N, обратное: P = M⁻¹×C mod N

Все операции выполняются в конечном поле вычетов по модулю N (длина алфавита), что гарантирует корректное обратное преобразование.
